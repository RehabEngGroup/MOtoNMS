Instructions

1) Run C3D2MAT.m

Output: All .c3d files in your InputData folder will be converted and saved in .mat format


Note

You need to have Matlab 32bit and install c3dserver (http://www.c3dserver.com/)
Input files folder path MUST include a folder named 'InputData'
It's necessary to convert .c3d files to be able to run DataProcessing step and StaticElaboration
Refer to MOtoNMS User Manual
